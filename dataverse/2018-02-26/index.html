19:30 Laerte48 Hi, would you please tell me how can I delete users? The section "manager user" doesn't have an optio to exclude a user... Thank you.
19:31 pdurbin Laerte48: hi! Deleting users isn't well supported but if you're confident that there is no data associated with that user, it's possible with a curl command.
19:32 pdurbin By data I mean stuff like `creator_id` in dvobject: http://phoenix.dataverse.org/schemaspy/latest/tables/dvobject.html (dataverses, datasets, and files all inherit from dvobject).
19:33 Laerte48 yes, I'm confident, because it is with two emails account that I've used for testing
19:33 pdurbin Laerte48: ok, I wrote about the curl command here: https://github.com/IQSS/dataverse/issues/1929#issuecomment-306545849
19:33 Laerte48 and there isn't any data
19:34 pdurbin curl -X DELETE http://localhost:8080/api/admin/authenticatedUsers/myself
19:34 pdurbin You would replace "myself" with the username.
19:34 Laerte48 Thank you very much
19:34 pdurbin sure
19:35 pdurbin Laerte48: if you want, please open a GitHub issue asking for a delete button on the user dashboard page. I forget what we call it.
19:36 pdurbin I guess we call it the "Manage Users" table in the dashboard: http://guides.dataverse.org/en/4.8.5/admin/user-administration.html#manage-users
19:36 Laerte48 ok, I'll see it.
