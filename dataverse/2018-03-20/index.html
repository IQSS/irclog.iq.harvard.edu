23:00 pdurbin This dataset of mine for example: https://dataverse.harvard.edu/dataset.xhtml?persistentId=doi:10.7910/DVN/TJCLKP
23:01 pdurbin Click "Metadata" and then "Export Metadata" and then "JSON".
23:01 pdurbin You should end up at this URL: https://dataverse.harvard.edu/api/datasets/export?exporter=dataverse_json&persistentId=doi%3A10.7910/DVN/TJCLKP
23:03 pdurbin I can get a list of file IDs like this: curl 'https://dataverse.harvard.edu/api/datasets/export?exporter=dataverse_json&persistentId=doi%3A10.7910/DVN/TJCLKP' | jq '.datasetVersion.files[].dataFile.id'
